# OOP
The concepts of Object Oriented Programming using C++
Here is a program in C++ , containing brief explaination and easy implementation of OOP concepts like:
-> Objects
-> Class
-> Encapsulation
-> Abstraction
-> Inheritance
-> Polymorphism
